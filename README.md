# Учебная программа по информатике и программированию для 5–11 классов

Программа разработана для преподавания в частной школе «Сократ» в условиях малых групп (около 10 человек в классе) с учебной нагрузкой 2 академических часа в неделю (рекомендуется проведение одного спаренного урока — «пары» продолжительностью 90 минут раз в неделю). Обучение организовано по триместрам (34 недели, 68 учебных часов в год).

## Главная цель программы
Формирование устойчивых и осознанных навыков эффективного и безопасного использования компьютерных технологий в современной цифровой среде. Приоритетом является проектная деятельность (Project-Based Learning, PBL), направленная на развитие самостоятельности, алгоритмического мышления и опыта командной работы.

---

## Основные принципы и методическая концепция

1. **Проект по итогам каждого триместра**
   Вместо классических контроческих работ каждый триместр завершается созданием и защитой индивидуального или группового (по 2–3 человека) проекта. Это позволяет применить полученные знания на практике и преодолеть страх чистого листа.
2. **Блочно-модульный подход**
   Один учебный год разделен на 34 блока. Каждый блок длится 2 академических часа и включает короткую теоретическую лекцию (15–20 минут), лабораторную практическую работу (40–45 минут) и время на разработку текущего проекта (20–25 минут).
3. **Развитие самостоятельности через ИТ-инструменты**
   Ученики с 5 класса знакомятся с профессиональными инструментами: разметка Markdown для лекций, консольный терминал для работы с файлами, Git/GitHub для совместных проектов и ведения портфолио.
4. **Гибридный стек ПО**
   Акцент сделан на кроссплатформенном и свободном программном обеспечении (LibreOffice, Python, Scratch, VS Code / Sublime Text), что позволяет выполнять домашние работы на любой операционной системе (Windows, MacOS, Linux).
5. **Интеграция систем ИИ**
   Искусственный интеллект рассматривается как инструмент помощника (ко-пилота). Ученики обучаются правильному составлению промптов (Prompt Engineering), фактчекингу ответов ИИ и этике использования нейросетей.

## Структурная матрица программы (5–11 классы)

| Класс | Триместр 1 | Триместр 2 | Триместр 3 |
|---|---|---|---|
| **5 класс** | Анатомия ПК, основы Word, Markdown и Obsidian. | Основы презентаций PowerPoint (Impress). Введение в Scratch. | Программирование в Scratch. Основы HTML5 и CSS3. |
| **6 класс** | Семантическая разметка HTML5 и CSS3. Растровая/векторная графика. | Углубленный Word (Writer) и продвинутый PowerPoint (Impress). | Работа с ИИ (фактчекинг, промпты). Основы видеомонтажа. |
| **7 класс** | Основы CLI (терминал Bash). Сетевая безопасность. | Облачные Google Docs и Sheets. LibreOffice Calc. | Введение в Python (модуль Turtle, переменные, циклы). |
| **8 класс** | Алгоритмы на Python. Работа с файлами. Базы знаний. | Веб-верстка: адаптивный дизайн и Flexbox (HTML/CSS). | Клиентский JS-интерактив. Сетевая безопасность (HTTPS, XSS). |
| **9 класс** | Анализ данных и подготовка к ОГЭ (Редактор ГОСТ Word, LaTeX). | Программирование под ОГЭ (Python, исполнитель Робот). | Комплексный веб-проект ОГЭ и решение вариантов. |
| **10 класс** | СУБД SQLite, SQL-запросы, Git/GitHub. Документы Word. | Сложные алгоритмы на Python (ООП, форматы CSV/JSON, Mail Merge). | Веб-панель (Dashboard), Pitch Deck презентация и ЕГЭ. |
| **11 класс** | Методы КЕГЭ (Задания 1–18). Старт ИТ-проекта. | Сложные алгоритмы КЕГЭ (19–27). Разработка бэкенда. | Репетиции КЕГЭ на ПК, Typst-верстка отчета и защита ИТ-проекта. |

## Структура репозитория

- `/README.md` — концепция и матрица программы.
- `/curriculum-matrix.md` — детальная сетка тем с разбивкой по часам и проектам.
- `/grades/` — поурочные планы (34 блока/пары на каждый класс):
  - `grade5.md` — 5 класс (компьютерная грамотность, Scratch);
  - `grade6.md` — 6 класс (офисные инструменты, графика, видео, markdown, ИИ);
  - `grade7.md` — 7 класс (терминал, сети, электронные таблицы, логика, Python Turtle);
  - `grade8.md` — 8 класс (программирование Python, веб-технологии HTML/CSS);
  - `grade9.md` — 9 класс (теоретическая информатика и подготовка к ОГЭ);
  - `grade10.md` — 10 класс (архитектура, СУБД SQLite, Git, структуры данных, ЕГЭ);
  - `grade11.md` — 11 класс (интенсивная подготовка к ЕГЭ, выпускной ИТ-проект).
