# Учебная программа по информатике и программированию для 5–11 классов

Программа разработана для преподавания в частной школе «Сократ» в условиях малых групп (около 10 человек в классе) с учебной нагрузкой 2 академических часа в неделю. Годовой курс содержит **68 учебных часов**, разбитых на поурочные занятия по 1 академическому часу (45 минут). Рекомендуется проводить занятия в формате спаренных уроков (34 пары по 90 минут раз в неделю). Обучение организовано по триместрам (34 недели в году).

## Главная цель программы
Развитие вычислительного мышления и навыков создания целостных цифровых продуктов. Изучение прикладного программного обеспечения, синтаксиса языков программирования и инструментов разработки рассматривается как средство достижения этой цели, а не как самоцель. Программа построена на непрерывной линии усложняющейся проектной деятельности (Project-Based Learning — PBL).

---

## Основные принципы и методическая концепция

1. **Непрерывная проектная линия**
   Вместо разрозненных контрольных работ каждый учебный год подчинен созданию сквозных цифровых продуктов. Проекты каждого триместра логически усложняют продукт, создавая непрерывную траекторию:
   * **5 класс:** Интерактивная игра (Scratch) -> Веб-сайт-презентация игры.
   * **6 класс:** Персональный сайт-визитка (HTML/CSS) -> Интеграция графики и видеопрезентации.
   * **7 класс:** Командные скрипты автоматики -> Программы на Python (Turtle) -> Автогенерация HTML-отчетов из CSV-данных.
   * **8 класс:** ООП-приложение на Python -> Адаптивный веб-клиент -> Динамическое веб-приложение (JS).
   * **9 класс:** Статистический проект с анализом данных (CSV/Excel) -> ОГЭ-тренажер алгоритмов -> Интерактивный ОГЭ-навигатор.
   * **10 класс:** Командная схема СУБД -> CRUD-приложение на Python -> Симулятор теории игр с GRID-дашбордом.
   * **11 класс:** Выпускной междисциплинарный проект -> Профиль безопасности CI -> Сборка отчетов в Typst и защита.

2. **Поурочное деление времени**
   Каждый урок (45 минут) включает краткое теоретическое введение (10 минут), практическую лабораторную работу (20 минут) и непосредственное конструирование элементов итогового проекта триместра (15 минут).

3. **Развитие алгоритмического мышления и анализа данных**
   Программа усиливает изучение фундаментальных понятий Computer Science: сложность алгоритмов ($O$-нотация), сортировки, поиск (бинарный), рекурсия, деревья решений и графы. Работа с данными начинается с 7 класса с использованием CSV-файлов, табличных процессоров Calc и визуализации графиков Matplotlib.

4. **Командная разработка и современный Git-инструментарий**
   GitHub используется как хранилище проектов с 5 класса (загрузка ZIP-архивов и файлов). Полноценная работа с Git (команды CLI, коммиты, история версий) вводится в 7 классе вместе с первыми программными проектами на Python. В 10-11 классах ученики изучают профессиональные инструменты командной разработки: ветвления веток, слияния, разрешение конфликтов, Kanban-планирование задач и практики Code Review.

5. **Прикладной ИИ как ко-пилот разработки**
   Использование генеративного ИИ введено в объеме **1 часа в конце каждого триместра** для решения сопутствующих задач: ускорения написания шаблонов верстки, поиска ошибок в логике, дебага и написания тестов. Ученики осваивают правила Prompt engineering и фактчекинг ответов ИИ.

| **8 класс** | Dictionaries, JSON-файлы, ООП-классы, Big-O Python. | Разметка Flexbox, Grid, адаптивность @media. | JS веб-программирование, DOM, события, валидация. |

## Структура репозитория

- `/README.md` — концепция и матрица программы (текущий файл).

  - `grade5.md` — 5 класс (компьютерная грамотность, Scratch);

  - `grade7.md` — 7 класс (терминал, сети, Python Turtle, разбор данных CSV, основы Git);
  - `grade8.md` — 8 класс (программирование Python, ООП, Flexbox, Grid, JavaScript);
  - `grade9.md` — 9 класс (аналитическая информатика, ОГЭ подготовка, Python/Matplotlib);
  - `grade10.md` — 10 класс (сетевой стек, Git-совместный, СУБД SQLite, ЕГЭ симуляции);
  - `grade11.md` — 11 класс (интенсивная подготовка к КЕГЭ, выпускной ИТ-проект, Typst).
