# Программа по информатике: 9 класс
**Нагрузка:** 2 часа в неделю (68 учебных часов в год, уроки по 1 академическому часу / 45 минут)

## Обзор программы
Курс 9 класса имеет двойной фокус: **подготовка к Основному государственному экзамену (ОГЭ) по информатике** по стандартам ФИПИ и создание **проекта с глубоким анализом данных**. Обучение базируется на проектном подходе (PBL), где теоретические концепции кодирования информации, логики и алгоритмики закрепляются при анализе реальных наборов данных (датасетов) с помощью Python и электронных таблиц.

### Проекты по триместрам:
* **Триместр 1:** Статистическое исследование (в Calc) на основе собранного опроса/датасета, экспортированное в структурированный формат CSV с диаграммами.
* **Триместр 2:** Интерактивный тренажер-пакет алгоритмов ОГЭ (решения задач 15.1 на КуМир исполнитель Робот и 15.2 на Python) со встроенными тестами для граничных значений.
* **Триместр 3:** "Сайт-навигатор ОГЭ" — веб-страница (HTML/CSS), на которой размещены сгенерированные скриптом Python отчеты и Matplotlib-графики анализа успеваемости, прохождения тестов и статистических закономерностей ОГЭ.

## Сценарии организации работы

### Сценарий 1: Индивидуальная траектория («Каждый сам»)
* Каждый ученик собирает свой уникальный датасет (например, архив погоды, характеристики гаджетов, личные спортивные показатели).
* Самостоятельно решает билеты ОГЭ в своей сборке скриптов.
* Индивидуально пишет Python скрипты анализа CSV, визуализирует и верстает отчетный сайт.
* **Результат:** Индивидуальный аналитический веб-ресурс и полная готовность к ОГЭ.

### Сценарий 2: Групповая траектория (команды из 2 человек)
* **Роль А — Аналитик данных (Data Analyst / Programmer):**
  * Отвечает за сбор и чистку данных, программирование формул в Calc, написание Python-кода обработки CSV-файлов, построение графиков Matplotlib и написания алгоритмов 15.2 на Python.
* **Роль Б — Технический писатель / Веб-разработчик (Technical Writer & Web Dev):**
  * Отвечает за составление ТЗ и README в Git, верстку отчетов в LaTeX/Word и итогового сайта-навигатора на HTML/CSS, написание алгоритмов Робота (15.1) в КуМир и общее тестирование.
