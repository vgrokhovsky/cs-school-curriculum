# Программа по информатике: 6 класс
**Нагрузка:** 2 часа в неделю (68 учебных часов в год, уроки по 1 академическому часу / 45 минут)

## Обзор программы
Курс 6 класса посвящен основам веб-компоновки и стилизации интерфейсов. Главной целью является развитие вычислительного мышления и навыков создания цифровых продуктов. На протяжении всего учебного года ученики работают над непрерывным учебным проектом — **собственным персональным сайтом-визиткой**. Конкретные технологии (HTML, CSS, графические редакторы, видеомонтаж) выступают в роли инструментов реализации проектного замысла.

### Проекты по триместрам:
* **Триместр 1:** Семантическая разметка (HTML5) страниц сайта-визитки.
* **Триместр 2:** Стилизация страниц сайта с помощью CSS3 (цвета, шрифты, отступы, Box Model).
* **Триместр 3:** Медианаполнение сайта (логотип в векторе, баннер в растре, видеопрезентация на веб-странице) и локальная публикация проекта.

---

## Сценарии организации работы

### Сценарий 1: Индивидуальная траектория («Каждый сам»)
* Каждый ученик верстает собственную веб-страницу.
* Самостоятельно пишет HTML и CSS файлы.
* Рисует и обрабатывает индивидуальные картинки в GIMP/Inkscape.
* Самостоятельно монтирует видео о себе и загружает на сайт.
* **Результат:** Индивидуальный персональный сайт с графикой и видео.

### Сценарий 2: Групповая траектория (команды по 2 человека)
* **Роль А — Веб-верстальщик (HTML/CSS Developer):**
  * Отвечает за чистоту HTML-кода, семантическую разметку, структуру тегов, связывание CSS и отладку стилей Box Model.
* **Роль Б — Медиа-дизайнер / Редактор контента (Media & Content Designer):**
  * Создает и подготавливает растровую и векторную графику, оптимизирует размеры изображений, делает раскадровку и монтирует видеоролик, пишет тексты для страниц.

## Поурочное планирование (68 уроков)

### Триместр 1. Семантическая разметка персонального сайта (24 часа)

#### Урок 1: Вводный урок. Что такое Интернет и веб-страницы
* **Теория:** Как работают сайты. Клиент-серверная модель (браузер и веб-сервер) на пальцах. Чему будем учиться в 6 классе.
* **Практика:** Создание структуры папок проекта `Grade6/my-site/`.
* **Сам:** Создание папки, настройка VS Code.
* **Группа:** Разработчик настраивает VS Code, Дизайнер готовит папку `assets/img/`.

#### Урок 2: Основы разметки HTML5
* **Теория:** Роль HTML. Понятие тегов, атрибутов и элементов. Структурный скелет страницы: `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`.
* **Практика:** Создание первого файла `index.html`, написание каркаса кода.
* **Сам/Группа:** Заполнение мета-информации страницы (тег `<title>`).

#### Урок 3: Теги текстовой структуры: заголовки
* **Теория:** Иерархия заголовков `<h1>`...`<h6>`. Основные правила их использования (один `<h1>` на страницу).
* **Практика:** Разметка заголовков для разделов сайта-визитки.
* **Сам:** Верстка заголовков «Главная», «Мои увлечения», «Моя учеба».
* **Группа:** Разработчик пишет код, Дизайнер вносит текстовые формулировки заголовков.

#### Урок 4: Разметка абзацев и форматирование текста
* **Теория:** Теги абзацев `<p>`, переноса строки `<br>`, горизонтальной линии `<hr>`.
* **Практика:** Наполнение разделов сайта связным текстом о себе.
* **Сам/Группа:** Использование тегов логического выделения `<strong>` и `<em>` для ключевых слов.

#### Урок 5: Создание списков в HTML5
* **Теория:** Маркированные списки `<ul>`, нумерованные списки `<ol>`, элементы списка `<li>`.
* **Практика:** Верстка списка своих хобби и любимых школьных предметов.
* **Сам/Группа:** Создание сложного вложенного списка учебных целей.

#### Урок 6: Разметка гиперссылок в HTML5
* **Теория:** Атрибут `href`. Разница между относительными путями (на другие файлы сайта) и абсолютными (внешние веб-ресурсы).
* **Практика:** Настройка ссылок со страницы `index.html` на внешние образовательные ресурсы.
* **Сам/Группа:** Реализация открытия внешних ссылок в новой вкладке (атрибут `target="_blank"`).

#### Урок 7: Создание многостраничной структуры сайта
* **Теория:** Архитектура многостраничного сайта. Главная страница (обычно `index.html`) и внутренние страницы.
* **Практика:** Создание страниц `hobbies.html` и `contacts.html`.
* **Сам/Группа:** Связывание 3 страниц сайта сквозными ссылками навигационного меню.

#### Урок 8: Интеграция изображений в HTML5
* **Теория:** Тег `<img src="..." alt="...">`. Важность атрибута `alt` для слепых пользователей и поисковиков. Разрешенные веб-форматы (`.png`, `.jpg`, `.svg`).
* **Практика:** Размещение на страницах сайта личной аватарки и картинок хобби.
* **Сам/Группа:** Определение размеров картинок в коде, исправление путей к медиафайлам.

#### Урок 9: Верстка базовых таблиц в HTML5
* **Теория:** Логика табличной верстки: `<table>`, `<tr>` (строка), `<td>` (ячейка), `<th>` (заголовок столбца).
* **Практика:** Создание таблицы «Мой список дел на неделю».
* **Сам/Группа:** Настройка сетки таблицы с помощью базовых HTML-атрибутов границ.

#### Урок 10: Объединение ячеек таблиц. Сложные структуры
* **Теория:** Атрибуты `colspan` (объединение колонок) и `rowspan` (объединение строк).
* **Практика:** Сборка сложной таблицы расписания уроков с объединенными парами.
* **Сам/Группа:** Верстка расписания уроков на неделю.

#### Урок 11: Семантические элементы HTML5
* **Теория:** Различие семантической и несемантической верстки. Зачем нужны теги `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`, `<aside>`.
* **Практика:** Преобразование разметки сайта с заменой несемантических структур на семантические.
* **Сам/Группа:** Переработка шапки и подвала сайта-визитки.

#### Урок 12: Специальные символы и комментарии в HTML
* **Теория:** Символы-мнемоники (`&nbsp;`, `&copy;`, `&lt;`, `&gt;`). Комментирование кода `<!-- комментарий -->`.
* **Практика:** Добавление знака копирайта и года разработки в подвал (`<footer>`) сайта.
* **Сам/Группа:** Добавление комментариев-подсказок для каждого логического блока страниц.

#### Урок 13: Введение в растровую графику для сайта. Редактор GIMP
* **Теория:** Основы растра: пиксель, глубина цвета, форматы картинок для веба (сжатие JPEG против прозрачности PNG).
* **Практика:** Обрезка (кадрирование) изображений под один формат для галереи сайта.
* **Сам/Группа:** Подготовка 3 картинок портфолио с одинаковой высотой и шириной.

#### Урок 14: Работа со слоями в графическом редакторе
* **Теория:** Смысл слоев. Прозрачный альфа-канал.
* **Практика:** Создание коллажа из нескольких картинок на прозрачном фоне.
* **Сам/Группа:** Создание простого графического баннера для шапки сайта-визитки.

#### Урок 15: Инструменты выделения и обрезки объектов в GIMP
* **Теория:** Выделение контуров. Размытие краев (растушевка).
* **Практика:** Аккуратное выделение объекта (например, своей фотографии) и удаление фона вокруг него.
* **Сам/Группа:** Экспорт фото с прозрачным фоном `.png` для вставки на сайт.

#### Урок 16: Цветокоррекция и сохранение веб-ассетов
* **Теория:** Цветовое пространство RGB. Яркость, контраст и насыщенность цифровых картинок.
* **Практика:** Цветовая подгонка скриншотов и фото под единое оформление. Оптимизация размера файлов (компрессия JPG).
* **Сам/Группа:** Сохранение оптимизированных картинок в папку проекта.

#### Урок 17: Создание форм ввода в HTML5. Базовые поля
* **Теория:** Тег `<form>`, поля ввода `<input type="text">`, кнопки `<button>`.
* **Практика:** Написание простейшей формы обратной связи на странице `contacts.html`.
* **Сам/Группа:** Добавление полей для ввода имени и почты пользователя.

#### Урок 18: Флажки и переключатели в HTML-формах
* **Теория:** Поля выбора `<input type="checkbox">` и переключатели `<input type="radio">`.
* **Практика:** Интеграция в контактную форму мини-опроса «Как вы узнали об этом сайте?».
* **Сам/Группа:** Сборка интерактивной формы опроса.

#### Урок 19: Кнопка отправки формы и текстовые области
* **Теория:** Тег `<textarea>`, типы кнопок `submit`/`reset` в формах.
* **Практика:** Реализация отправки формы имитационным методом. Добавление поля «Сообщение».
* **Сам/Группа:** Верстка завершенного блока контактов.

#### Урок 20: Валидация HTML-кода
* **Теория:** Правила чистого веб-кода. Понятие спецификации W3C. Инструменты онлайн/офлайн валидации HTML-кода.
* **Практика:** Проверка написанных страниц сайта-визитки на валидность, исправление незакрытых тегов и опечаток.
* **Сам/Группа:** Устранение предупреждений валидатора.

#### Урок 21: Применение ИИ при разметке веб-страниц (1 час ИИ)
* **Теория:** Как использовать ИИ в веб-разработке. Промпты для поиска опечаток в HTML-коде.
* **Практика:** Передача страниц сайта ИИ-помощнику для выявления логических ошибок разметки, опечаток в атрибутах и отсутствия тегов `alt`.
* **Сам/Группа:** Очистка кода сайта на основе рекомендаций ИИ.

#### Урок 22: Фактчекинг текстового наполнения сайта
* **Теория:** Проверка достоверности данных. Безопасность упоминания личной информации на публичных страницах.
* **Практика:** Поиск и удаление слишком личных данных (точные домашние адреса, номера телефонов родителей) с макета сайта-визитки.
* **Сам/Группа:** Написание безопасного и грамотного описания о себе.

#### Урок 23: Подготовка к защите структуры сайта-визитки
* **Теория:** Как устроена сдача первого этапа разработки. Проверка работоспособности относительных ссылок меню.
* **Практика:** Исправление битых ссылок при переходе между страницами.
* **Сам/Группа:** Составление чек-листа готовности HTML-структуры.

#### Урок 24: Защита проектов: HTML-каркас сайта-визитки (Защита проектов)
* **Теория:** Презентация макета сайта коллегам.
* **Практика:** Демонстрация веб-страниц без стилей в браузере, проверка валидаторами. Взаимооценка.
* **Результат триместра:** Набор семантически верных веб-страниц с готовым контентом и таблицами.

### Триместр 2. Стилизация сайта с помощью CSS3 (24 часа)

#### Урок 25: Введение в стили CSS3. Селекторы тегов
* **Теория:** Понятие каскадных таблиц стилей. Способы подключения CSS (встроенные, внутренние, внешние стили). Структура правила: селектор, свойства, значения.
* **Практика:** Создание внешнего файла `styles/main.css` и изменение цвета фона страниц сайта.
* **Сам/Группа:** Стилизация шрифтов всех текстовых страниц.

#### Урок 26: Цвета и фоны в CSS3
* **Теория:** Способы кодирования цвета (названия, RGB/RGBA, HEX коды).
* **Практика:** Разработка гармоничной палитры цветов сайта-визитки с помощью онлайн-генераторов палитр.
* **Сам/Группа:** Цвет текста, фона страниц, границ таблиц.

#### Урок 27: Основы веб-типографики в CSS3
* **Теория:** Семейства шрифтов, безопасные системные шрифты. Свойства `font-size`, `font-weight`, `line-height`, `text-align`.
* **Практика:** Настройка читаемости текстов сайта, увеличение межстрочного расстояния.
* **Сам/Группа:** Стилизация заголовков страниц и выделенного текста.

#### Урок 28: Использование веб-декораций в CSS3
* **Теория:** Свойства `text-decoration` (подчеркивание), `text-transform` (заглавные), `text-shadow` (тени текста).
* **Практика:** Устранение стандартного синего подчеркивания у ссылок меню сайта.
* **Сам/Группа:** Создание красивых эффектов тени для заголовка сайта.

#### Урок 29: Селекторы классов в CSS
* **Теория:** Отличие селекторов тегов от селекторов классов (`.class`). Зачем нужны классы в верстке.
* **Практика:** Разметка и стилизация разделов «Срочные дела» и «Обычные дела» в таблице с помощью отдельных классов.
* **Сам/Группа:** Применение классов для раскрашивания четных строк таблицы.

#### Урок 30: Селекторы идентификаторов (ID selectors)
* **Теория:** Уникальные селекторы идентификаторов (`#id`). Отличие классов от идентификаторов (ID только один на страницу).
* **Практика:** Стилизация главного блока шапки сайта по уникальному ID.
* **Сам/Группа:** Настройка стилей для футера с ID.

#### Урок 31: Блочная модель CSS (Box Model)
* **Теория:** Физика блоков в веб: ширина (`width`), высота (`height`), границы (`border`).
* **Практика:** Создание рамки вокруг фотографии автора с закруглением уголков (`border-radius`).
* **Сам/Группа:** Оформление рамок для таблиц и форм.

#### Урок 32: Блочная модель CSS: поля и отступы
* **Теория:** Внутренние отступы (`padding`) и внешние поля (`margin`). Влияние полей на выравнивание объектов.
* **Практика:** Создание отступов внутри карточек товаров/увлечений, чтобы текст не прилипал к краям рамки.
* **Сам/Группа:** Центрирование контентного блока сайта с помощью `margin: 0 auto;`.

#### Урок 33: Настройка обтекания элементов: свойство float
* **Теория:** Позиционирование элементов. Обтекание картинок текстом с помощью `float: left` / `right`. Сброс обтекания (`clear`).
* **Практика:** Верстка текстового блока с картинкой, обтекаемой текстом с аккуратным отступом вбок.
* **Сам/Группа:** Верстка боковой панели (aside) с использованием обтекания.

#### Урок 34: Разница блочных и строчных элементов в CSS
* **Теория:** Блочные элементы (`display: block`) против строчных (`display: inline`). Преобразования блоков с помощью `display`.
* **Практика:** Преобразование списков навигационного меню `<ul>` в горизонтальную строчную панель ссылок.
* **Сам/Группа:** Построение горизонтального меню сайта-визитки.

#### Урок 35: Псевдоклассы интерактивности CSS
* **Теория:** Динамические селекторы событий: `:hover` (наведение мыши), `:active` (момент клика). Плавность анимации (`transition`).
* **Практика:** Создание интерактивных кнопок меню: при наведении мыши цвет фона плавно меняется.
* **Сам/Группа:** Добавление эффектов наведения на скриншоты.

#### Урок 36: Стилизация форм обратной связи
* **Теория:** Оформление полей ввода, фокусы полей (`:focus`).
* **Практика:** Верстка формы: плавная смена рамки поля при вводе имени, стилизация кнопки отправки.
* **Сам/Группа:** Наведение красоты в блоке контактов.

#### Урок 37: Введение в векторную графику для сайта. Inkscape
* **Теория:** Вектор против растра. Плюсы SVG-формата для сайтов (масштабирование без пикселей).
* **Практика:** Создание простых векторных фигур в Inkscape.
* **Сам/Группа:** Разработка дизайна логотипа сайта-визитки.

#### Урок 38: Создание логотипа: операции с контурами в Inkscape
* **Теория:** Узлы, кривые. Булевы операции сложения и вычитания фигур.
* **Практика:** Рисование красивой иконки (логотипа) вычитанием кругов и заливкой градиентом.
* **Сам/Группа:** Сохранение готового логотипа в формате `.svg`.

#### Урок 39: Шрифты от сторонних сервисов (Google Fonts / ЛТ)
* **Теория:** Подключение оригинальных шрифтов через тег `<link>` в HTML.
* **Практика:** Выбор шрифта (например, Onest / Cyrillic) и подключение его к сайту-визитке.
* **Сам/Группа:** Тонкая настройка начертаний шрифта на страницах.

#### Урок 40: Градиенты и фоны CSS3
* **Теория:** Линейные (`linear-gradient`) и радиальные градиенты.
* **Практика:** Заполнение фоновых панелей страниц стильными градиентными переходами.
* **Сам/Группа:** Создание градиентного хедера сайта.

#### Урок 41: Тени блоков (box-shadow)
* **Теория:** Свойство `box-shadow`: сдвиг по X/Y, размытие, цвет тени.
* **Практика:** Добавление эффекта «парения» карточкам хобби при наведении с помощью плавного появления тени.
* **Сам/Группа:** Придание объема элементам интерфейса сайта.

#### Урок 42: Проектирование макета: Введение во Flexbox
