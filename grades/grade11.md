# Программа по информатике: 11 класс
**Нагрузка:** 2 часа в неделю (34 блока / пары по 90 минут, всего 68 часов)

## Обзор программы
Курс 11 класса посвящен углубленной подготовке к Единому государственному экзамену (ЕГЭ) по информатике в компьютерной форме (КЕГЭ) и выполнению итогового междисциплинарного ИТ-проекта. Обучение базируется на продвинутых алгоритмах на Python, обработке больших объемов числовых и текстовых данных, основах математического моделирования и теории игр.

## Распределение часов по триместрам
* **Триместр 1:** Методы КЕГЭ и программирование базовых задач (Задания 1–18) (12 пар, 24 часа)
* **Триместр 2:** Олимпиадные алгоритмы в КЕГЭ (Задания 19–27) (12 пар, 24 часа)
* **Триместр 3:** Пробные экзамены и выпускной ИТ-проект (10 пар, 20 часов)

## Поурочное планирование (34 спаренных урока)

### Триместр 1. Методы КЕГЭ и программирование базовых задач (Задания 1–18)

#### Блок 1: Системы счисления в ЕГЭ (Задание 14)
* **Теория:** Позиционные системы счисления. Арифметика в различных базисах. Поиск неизвестных цифр в выражениях с большими степенями.
* **Практика:** Написание скриптов на Python для быстрого перебора неизвестных цифр (например, в выражении $123x5_{15} + 1x234_7$) и нахождения правильных решений.
* **Проект:** Инициализация итогового проекта: выбор темы («Мобильное приложение», «Телеграм-бот», «Веб-сервис», «Исследовательский проект по анализу данных»).

#### Блок 2: Таблицы истинности и логические выражения (Задание 2)
* **Теория:** Таблицы истинности для 3–4 переменных. Логическое следование (импликация), эквивалентность.
* **Практика:** Построение таблиц истинности с помощью циклов перебора `product` из модуля `itertools`. Подгонка шаблона под фрагмент таблицы в условии задачи.
* **Проект:** Разработка блок-схемы архитектуры и требований к междисциплинарному выпускному проекту.

#### Блок 3: Поиск информации по реляционным базам данных (Задание 3)
* **Теория:** Анализ многотабличных баз данных в Excel. Связывание данных по первичному ключу.
* **Практика:** Применение функции `ВПР` (VLOOKUP) и продвинутых фильтров для решения комплексных складских и финансовых задач ЕГЭ.
* **Проект:** Проектирование схемы данных для выпускного ИТ-проекта.

#### Блок 4: Адресация в сетях TCP/IP (Задание 13)
* **Теория:** Структура IP-адресов. Сетевые маски. Понятие адреса сети. Определение количества адресов в подсети.
* **Практика:** Решение задач на поиск возможных IP-адресов и масок с использованием побитовых операций в Python (библиотека `ipaddress`).
* **Проект:** Развертывание локального окружения для разработки проекта.

#### Блок 5: Кодирование звука и графики. Передача данных (Задание 7)
* **Теория:** Формулы информационного объема звука ($I = \beta \cdot f \cdot t \cdot M$) и графического файла ($I = \text{разрешение} \cdot \text{глубина цвета}$). Скорость передачи пакетов.
* **Практика:** Решение расчетных аналитических задач ЕГЭ на сжатие данных при передаче по каналам связи.
* **Проект:** Подбор необходимых библиотек и инструментов для проекта.

#### Блок 6: Комбинаторика в ЕГЭ на Python (Задание 8)
* **Теория:** Комбинаторные объекты: размещения, сочетания, перестановки. Правила сложения и умножения вариантов.
* **Практика:** Использование генераторов комбинаторных объектов `permutations`, `combinations`, `product` из модуля `itertools` для поиска количества слов по сложным алфавитным правилам.
* **Проект:** Программирование ядра бизнес-логики выпускного ИТ-проекта.

#### Блок 7: Программирование исполнителя Редактор (Задание 12)
* **Теория:** Логика работы встроенного алгоритма замены подстрок.
* **Практика:** Моделирование работы Редактора на Python с использованием строковых методов `.replace()`. Поиск закономерностей в циклических строках.
* **Проект:** Git-коммиты первой рабочей версии ядра выпускного проекта.

#### Блок 8: Анализ программ с циклами и ветвлениями (Задание 15)
* **Теория:** Определение множеств чисел, отрезков на координатных осях и побитовых операций, обеспечивающих тождественную истинность логического выражения.
* **Практика:** Написание алгоритмов нахождения границ интервала $A$ через оптимизационный перебор.
* **Проект:** Разработка интерфейса пользователя для ИТ-проекта.

#### Блок 9: Алгоритмы рекурсивных вычислений (Задание 16)
* **Теория:** Рекуррентные соотношения. Оптимизация хвостовой рекурсии, избегание дублирующих вычислений с помощью кэширования (`functools.lru_cache`).
* **Практика:** Расчет значений функций при больших аргументах (до $N=10^6$), математическое сокращение рекурсивных формул.
* **Проект:** Связывание фронтенда/интерфейса выпускного проекта с бэкенд логикой.

#### Блок 10: Поиск путей на графах (Задание 1)
* **Теория:** Соответствие весовой матрицы и графической структуры ненаправленного графа.
* **Практика:** Решение аналитических задач на сопоставление вершин графа и строк таблицы. Реализация алгоритма Дейкстры на Python.
* **Проект:** Проведение промежуточного тестирования функционала выпускного проекта.

#### Блок 11: Анализ числовых последовательностей (Задание 17)
* **Теория:** Скорость обработки массивов данных. Линейные проходы.
* **Практика:** Обработка файлов, содержащих массивы до 10 000 целых чисел. Поиск пар элементов, сумма которых кратна заданному числу, с оптимизацией $O(N)$.
* **Проект:** Интеграция базы данных в выпускной проект.

#### Блок 12: Динамическое программирование: Робот-сборщик монет (Задание 18)
* **Теория:** Двумерное динамическое программирование в электронных таблицах Calc. Учет стенок и ограничений на перемещение робота.
* **Практика:** Построение формул расчета максимальной и минимальной денежной суммы, собираемой Роботом в лабиринтах разного размера.
* **Проект:** Подготовка демо-презентации промежуточных результатов проекта.
