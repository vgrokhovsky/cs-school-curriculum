# Программа по информатике: 11 класс
**Нагрузка:** 2 часа в неделю (68 учебных часов в год, уроки по 1 академическому часу / 45 минут)

## Обзор программы
Курс 11 класса посвящен завершающему этапу среднего общего образования: **углубленной подготовке к Компьютерному ЕГЭ (КЕГЭ) по информатике** на языке Python и завершению **выпускного междисциплинарного ИТ-проекта** (индивидуального или в мини-группах). Вычислительное мышление и практические навыки разработки цифровых продуктов достигают здесь уровня профессионального интерна (junior developer). Ученики решают сложные задачи анализа последовательностей, теории игр и динамического программирования, применяют концепции безопасного программирования (Secure Coding) и документируют свои ИТ-решения с использованием профессиональной системы верстки Typst.

### Проекты по триместрам:
* **Триместр 1:** Работающий рабочий прототип (MVP) выпускного ИТ-проекта (например, веб-приложение с СУБД, Telegram-бот с ИИ, аналитическая система прогнозирования).
* **Триместр 2:** Полная бэкенд-реализация проекта с тестами производительности и безопасности, репозиторий на GitHub с настроенным CI и релизами.
* **Триместр 3:** Готовый междисциплинарный ИТ-проект с встроенной аналитикой, а также пояснительная записка (отчет разработчика) профессионально сверстанная в системе Typst, и защита на школьной ИТ-конференции.

## Сценарии организации работы

### Сценарий 1: Индивидуальная траектория («Каждый сам»)
* Каждый ученик выбирает отдельную сложную задачу из смежных областей (физика, математика, биология, лингвистика) и решает её через разработку ИТ-продукта.
* Самостоятельно пишет бэкенд, структурирует базу данных, верстает интерфейс.
* Еженедельно проходит тренировочные варианты КЕГЭ в индивидуальном темпе.
* **Результат:** Завершенный индивидуальный ИТ-проект и высокие баллы на ЕГЭ.

### Сценарий 2: Групповая траектория (команды из 2 человек)
Распределение ролей для профессионального выпуска продукта:
* **Роль А — Системный архитектор / Бэкенд-разработчик (Backend & Database Engineer):**
  * Отвечает за разработку ядра программы на Python, интеграцию СУБД (PostgreSQL/SQLite), проектирование API, оптимизацию алгоритмов до $O(N)$ для работы со 100 000+ записей и прохождение задач 26-27 КЕГЭ.
* **Роль Б — Веб-разработчик / Дизайнер документации (Frontend Engineer & Technical Writer):**
  * Разрабатывает интерфейсную часть проекта (HTML/CSS/JS или GUI на Python), верстает пояснительные отчеты в издательской системе Typst, отвечает за аудит безопасности зависимостей (`pip-audit`) и проведение код-ревью.
