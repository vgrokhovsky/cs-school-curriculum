# Программа по информатике: 7 класс
**Нагрузка:** 2 часа в неделю (34 блока / пары по 90 минут, всего 68 часов)

## Обзор программы
Курс 7 класса переходит к более глубоким техническим аспектам ИТ. В первой части ученики осваивают текстовый интерфейс операционной системы через консольный терминал и разбираются в физическом и логическом устройстве компьютерных сетей. Во второй части изучаются электронные таблицы для аналитики данных. Завершается год введением в текстовое программирование на Python с использованием графической библиотеки Turtle.

## Распределение часов по триместрам
* **Триместр 1:** Командная строка CLI и компьютерные сети (12 пар, 24 часа)
* **Триместр 2:** Анализ данных в электронных таблицах LibreOffice Calc (12 пар, 24 часа)
* **Триместр 3:** Введение в программирование на Python: Turtle-графика (10 пар, 20 часов)

## Поурочное планирование (34 спаренных урока)

### Триместр 1. Командная строка CLI и компьютерные сети

#### Блок 1: Графический интерфейс против текстового. Понятие терминала
* **Теория:** Интерфейс командной строки (CLI) и графический интерфейс (GUI). Почему профессионалы выбирают CLI. Основные оболочки (Bash, Zsh, PowerShell).
* **Практика:** Запуск терминала. Базовая справочная команда `man` / `--help`. Команда `clear` и история `history`.
* **Проект:** Начало работы над «Скриптом упорядочивания файлов» — создание папки проекта.

#### Блок 2: Навигация в файловой системе через терминал
* **Теория:** Абсолютные и относительные пути. Обозначения текущей директории (`.`) и родительской директории (`..`), домашнего каталога (`~`).
* **Практика:** Перемещение по диску с помощью команды `cd`. Просмотр содержимого папок с помощью `ls` (ключи `-l`, `-a`, `-h`).
* **Проект:** Исследование структуры папок проекта через CLI.

#### Блок 3: Управление файлами и папками в CLI
* **Теория:** Синтаксис команд создания, копирования, перемещения и удаления. Опасность команды удаления в консоли.
* **Практика:** Создание папок (`mkdir`), файлов (`touch`), копирование (`cp`), перемещение (`mv`) и безопасное удаление (`rm`).
* **Проект:** Создание древовидной структуры каталогов для сортировки файлов.

#### Блок 4: Просмотр содержимого файлов и текстовые редакторы в терминале
* **Теория:** Способы быстрого чтения файлов без графической среды. Знакомство с консольным редактором Nano.
* **Практика:** Просмотр файлов командами `cat`, `less`, `head`, `tail`. Создание и редактирование документа в Nano.
* **Проект:** Написание справочного файла `instructions.txt` внутри проекта.

#### Блок 5: Потоки ввода-вывода и конвейеры (Pipelines)
* **Теория:** Стандартный ввод (stdin), стандартный вывод (stdout). Перенаправление потоков (`>`, `>>`). Понятие конвейера (`|`).
* **Практика:** Поиск определенной строки в выводе команды с помощью `grep`. Слияние вывода нескольких команд в один лог-файл.
* **Проект:** Создание текстового отчета о файлах папки с автоматической записью в файл.

#### Блок 6: Создание простейших скриптов Bash
* **Теория:** Заголовок скрипта `#!/bin/bash`. Права доступа к файлам и команда изменения прав `chmod +x`.
* **Практика:** Написание скрипта, который одной командой делает бэкап папки проектов (создает папку-копию с текущей датой).
* **Проект:** Первая заготовка скрипта сортировки файлов.

#### Блок 7: Переменные и аргументы командной строки в скриптах
* **Теория:** Переменные среды в Bash. Аргументы `$1`, `$2` как способ параметризации работы скрипта.
* **Практика:** Доработка скрипта бэкапа: теперь он принимает имя папки бэкапа как аргумент при запуске.
* **Проект:** Параметризация скрипта сортировки (указание целевой папки).

#### Блок 8: Логические ветвления в Bash-скриптах
* **Теория:** Проверка существования файлов и папок: условия `if [ -d ... ]` и `if [ -f ... ]`. Условные операторы Bash.
* **Практика:** Написание скрипта очистки временных файлов с проверкой, существует ли папка с мусором.
* **Проект:** Добавление проверок безопасности в скрипт упорядочивания файлов.

#### Блок 9: Законченный скрипт упорядочивания файлов по расширениям
* **Теория:** Использование циклов `for` для обхода файлов в папке. Маски файлов (например, `*.jpg`, `*.pdf`).
* **Практика:** Написание цикла, проходящего по всем файлам директории и распределяющего картинки в папку "Images", документы в "Docs" и т. д.
* **Проект:** Сборка и тестирование итогового Bash-скрипта.

#### Блок 10: Устройство компьютерных сетей и стек TCP/IP
* **Теория:** Как компьютеры понимают друг друга. IP-адресация (IPv4 и IPv6). Маска подсети. LAN (локальная сеть) и WAN (глобальная сеть). Маршрутизаторы.
* **Практика:** Команда `ip a` или `ifconfig`. Определение локального IP-адреса и шлюза компьютера.
* **Проект:** Размещение результирующего скрипта в сетевой папке.

#### Блок 11: Работа DNS и диагностика сети
* **Теория:** Что такое доменное имя и служба DNS. Маршрутизация пакетов данных. Порты протоколов.
* **Практика:** Сетевая диагностика утилитами `ping` (соединение), `traceroute` (путь прохождения пакета), `nslookup` (проверка доменного имени).
* **Проект:** Проверка сетевой связности с помощью написанного скрипта диагностики.

#### Блок 12: Защита проектов: Скрипт автоматизации
* **Теория:** Защита работ в терминале. Анализ логов работы Bash-скриптов.
* **Практика:** Демонстрация скриптов сортировки в действии на "замусоренных" папках. Взаимное оценивание.
* **Проект:** Сдача Bash-проекта «Упорядочиватель файлов».

### Триместр 2. Анализ данных в электронных таблицах LibreOffice Calc

#### Блок 13: Понятие электронных таблиц. Интерфейс LibreOffice Calc
* **Теория:** Для чего используются электронные таблицы. Строки, столбцы, ячейки, диапазоны данных. Типы данных (текст, числа, даты).
* **Практика:** Создание книги Calc. Заполнение таблицы данных вручную. Настройка автозаполнения (числовые ряды, дни недели).
* **Проект:** Запуск проекта «Модель планирования личных/семейных финансов».

#### Блок 14: Формулы и простая математика в ячейках
* **Теория:** Правило начала формулы со знака `=`. Арифметические действия. Порядок вычислений.
* **Практика:** Создание товарного чека с расчетом цены, количества, суммы и итоговой стоимости покупки.
* **Проект:** Разработка листа «Доходы и Расходы» с формулами суммирования за день.

#### Блок 15: Относительная и абсолютная адресация ячеек. Знак `$`
* **Теория:** Что происходит при копировании формулы? Ссылка на конкретную статичную ячейку (абсолютная ссылка).
* **Практика:** Расчет стоимости товаров в валюте с привязкой к одной ячейке «Курс доллара» при помощи знака доллара `$`.
* **Проект:** Реализация пересчета ежемесячных расходов с учетом коэффициента инфляции на листе проекта.

#### Блок 16: Встроенные статистические функции
* **Теория:** Вызов встроенных функций. Понятие аргументов функции. Функции: `СУММ` (SUM), `СРЗНАЧ` (AVERAGE), `МИН` (MIN), `МАКС` (MAX).
* **Практика:** Расчет среднего балла успеваемости класса, поиск лучшего и худшего ученика по баллам.
* **Проект:** Добавление статистического анализа расходов по месяцам в финансовую модель.

#### Блок 17: Логические вычисления. Условие `ЕСЛИ` (IF)
* **Теория:** Логическое выражение. Конструкция ветвления ячейки: `=ЕСЛИ(условие; значение_если_истина; значение_если_ложь)`.
* **Практика:** Расчет скидки на товар: если сумма покупки более 5000 рублей, то скидка 10%, иначе скидки нет.
* **Проект:** Настройка предупреждающих сообщений о дефиците бюджета при превышении лимита расходов.

#### Блок 18: Условное форматирование данных
* **Теория:** Зачем нужно выделять цветом ячейки автоматически? Правила форматирования на основе значений.
* **Практика:** Цветовая индикация оценок учеников (двойки — красные, пятерки — зеленые).
* **Проект:** Автоматическое выделение цветом статей расходов с критическим перерасходом.

#### Блок 19: Сортировка и фильтрация списков данных
* **Теория:** Подготовка таблицы под базу данных. Сортировка по возрастанию/убыванию, многоуровневая сортировка. Фильтр по критерию.
* **Практика:** Анализ таблицы из 100 строк с перечнем стран мира (извлечение топ-10 стран по населению, фильтр по материкам).
* **Проект:** Анализ категорий трат (например, вывод только ИT-покупок за год).

#### Блок 20: Визуализация данных. Диаграммы и графики
* **Теория:** Типы графиков: круговая диаграмма (доли от общего), гистограмма (сравнение), график (динамика во времени).
* **Практика:** Построение температурного графика за неделю и круговой диаграммы состава атмосферы Земли.
* **Проект:** Построение круговой диаграммы расходов семьи по категориям и графика изменения остатка средств.

#### Блок 21: Функции подсчета по условию `СЧЁТЕСЛИ` (COUNTIF) и `СУММЕСЛИ` (SUMIF)
* **Теория:** Статистика выборочных данных. Как просуммировать только строго определенные траты.
* **Практика:** Анализ журнала посещаемости: подсчет количества пропусков по болезни для каждого ученика.
* **Проект:** Создание сводной аналитической таблицы «Расходы по категориям за год» с помощью `СУММЕСЛИ`.

#### Блок 22: Защита листов и совместная работа
* **Теория:** Как скрыть ценные формулы от случайного удаления. Защита паролем. Экспорт в формат `.csv`.
* **Практика:** Совместное редактирование файла таблицы Calc несколькими пользователями.
* **Проект:** Фиксация финансовой модели, защита формул от изменения пользователем.

#### Блок 23: Оптимизационные задачи: сценарии «Что если...»
* **Теория:** Поиск решений. Изменение исходных параметров для достижения планируемого результата.
* **Практика:** Задача: рассчитать, какую сумму ежемесячно нужно откладывать под процент для покупки велосипеда через полгода.
* **Проект:** Настройка сценария накопления на отпуск в рамках личного бюджета.

#### Блок 24: Защита финансовых моделей (Презентация выводов)
* **Теория:** Представление результатов аналитической работы перед аудиторией.
* **Практика:** Защита финансовой модели. Обоснование структуры расходов и мер экономии на основе данных таблицы.
* **Проект:** Сдача готовой работы «Модель личных финансов».

### Триместр 3. Введение в программирование на Python: Turtle-графика

#### Блок 25: Язык Python. Установка среды и знакомство с IDLE / VS Code
* **Теория:** Почему Python популярен. Интерпретируемые языки. Понятие REPL (интерактивный режим) и Скрипт (программный код).
* **Практика:** Запуск интерактивной консоли Python. Вывод сообщений функцией `print()`. Простейшая арифметика в консоли.
* **Проект:** Выбор темы генеративного арта (узор, мозаика или фрактал) для финального проекта.

#### Блок 26: Импорт модулей. Основы библиотеки Turtle (Черепашка)
* **Теория:** Как подключить библиотеку (`import turtle`). Движение вперед `forward()`, назад `backward()`, повороты `left()`, `right()`.
* **Практика:** Написание программы рисования правильного квадрата с использованием пошаговых инструкций Черепашки.
* **Проект:** Настройка окна вывода графики и скорости отрисовки (`speed`).

#### Блок 27: Переменные в Python и ввод данных
* **Теория:** Использование переменных. Оператор присваивания `=`. Запрос данных у пользователя функцией `input()`. Преобразование типов `int()`.
* **Практика:** Запрос у пользователя длины стороны квадрата и отрисовка квадрата соответствующего размера.
* **Проект:** Добавление интерактивного меню настройки цвета и размера узора перед началом рендеринга.

#### Блок 28: Циклы `for` и функция `range()` в Python
* **Теория:** Избавление от дублирования кода. Цикл `for i in range(N):`. Логика отступов в Python (блоки кода).
* **Практика:** Переписывание кода рисования квадрата с использованием цикла `for`. Рисование правильного 6-угольника и 8-угольника.
* **Проект:** Написание циклического алгоритма генерации базовых геометрических элементов.

#### Блок 29: Разноцветная Черепашка. Списки цветов и случайные числа
* **Теория:** Списки в Python (базовое понятие). Подключение модуля `random`. Случайный выбор элемента функцией `random.choice()`.
* **Практика:** Создание скрипта, рисующего спираль, где каждая грань меняет цвет случайным образом из списка.
* **Проект:** Разработка палитры градиентных переходов для узора.

#### Блок 30: Функции в Python. Создание собственных графических блоков
* **Теория:** Зачем нужны функции? Объявление функции через ключевое слово `def имя_функции(параметры):`. Аргументы функций.
* **Практика:** Написание функции `draw_star(size, color)` для рисования звезды любого размера в любой точке экрана.
* **Проект:** Группировка повторяющихся элементов узора в универсальные функции.

#### Блок 31: Алгоритмические ветвления `if - else` в Python
* **Теория:** Сравнение значений в Python (`==`, `>`, `<`, `!=`). Условные конструкции `if`, `elif`, `else`.
* **Практика:** Черепашка меняет направление или цвет в зависимости от текущей координаты X на экране (пересечение золотой середины).
* **Проект:** Добавление логики ветвления для изменения узора во внутренних кругах / границах листа.

#### Блок 32: Обработка событий клавиатуры и мыши
* **Теория:** Событийно-ориентированное программирование. Регистрация функций-слушателей (`onscreenclick`, `onkey`).
* **Практика:** Создание мини-приложения «Рисовалка»: ученик водит черепашку стрелками клавиатуры и оставляет след на холсте.
* **Проект:** Добавление возможности перегенерации узора по нажатию клавиши "R" пробела.

#### Блок 33: Тестирование алгоритмов и устранение ошибок
* **Теория:** Чтение сообщений об ошибках в консоли Python (SyntaxError, NameError, TypeError). Логическое чтение трассировок стека.
* **Практика:** Поиск и исправление специально допущенных ошибок в анимационном коде.
* **Проект:** Финальная чистка кода узора, рефакторинг названий переменных под правила PEP8.

#### Блок 34: Выставка генеративного искусства (Защита проектов)
* **Теория:** Форматы обмена Python кодом.
* **Практика:** Демонстрация генерации узоров в реальном времени. Обсуждение заложенных математических пропорций.
* **Проект:** Презентация итоговой программы «Генератор геометрических узоров на Python».
